**GNC** is **N**ot **C**. It is intended for a better and more effective c language.

## Grammar Features

- [x] int main() ...
- [x] declare local int variable
- [x] unary operation  
- [x] binary operation
- [x] conditional statement
- [x] loop statement
- [x] scope 
- [x] function
- [x] global variable
- [ ] array
- [ ] pointer
- [ ] more types
- [ ] string
- [ ] struct
- [ ] struct with function
